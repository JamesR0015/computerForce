NOTE TO STUDENTS:
This folder will serve as your project folder. All assessment documents, webpage files and media assets will be saved here.

SYSTEM REQUIREMENTS:
PHP and mySQL running in XAMPP, MAMP, LAMPP, etc.

XAMPP Installation Instructions:
Windows:
https://www.wikihow.com/Install-XAMPP-for-Windows

Mac(For intel users only. For M1 chip, please install MAMP instead):
https://www.intelliwolf.com/how-to-install-xampp-step-by-step/

SETUP INSTRUCTIONS:
1. Rename the folder computerForceStarterKit to computerForce.
2. Save this computerForce folder in htdocs.
3. In XAMPP, start APACHE and MYSQL.
4. In phpmyadmin site, import computerForce.sql located at the database folder.
   This will create dbComputerForce database and customer table. To go to
   phpmyadmin, enter: localhost/phpmyadmin.
5. To run this website, enter localhost/computerForce.
